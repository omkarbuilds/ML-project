# Resume Screening System

A Machine Learning–based Resume Screening System that automatically analyzes and classifies resumes to help shortlist candidates efficiently based on job requirements.

> This project is part of my Machine Learning learning journey, focusing on the practical application of NLP techniques.

---

## 📌 Project Overview

Manual resume screening is time-consuming and often inconsistent.  
This project automates the resume screening process using **Machine Learning and Natural Language Processing (NLP)** to improve efficiency and consistency in candidate shortlisting.

---

## 🚀 Features

- Resume text preprocessing (cleaning and normalization)
- Feature extraction using **TF-IDF**
- **Random Forest classifier** for resume classification
- Automated resume screening and prediction
- Scalable for different job roles

---

## 🛠️ Tech Stack

- **Programming Language:** Python  
- **Machine Learning Algorithm:** Random Forest  
- **NLP Techniques:** TF-IDF, NLTK  
- **Libraries:** Scikit-learn, Pandas, NumPy, Matplotlib, Seaborn  
- **Environment:** Jupyter Notebook  

---

## 📂 Dataset

- Dataset sourced from **Kaggle**
- File used: `ai_resume_screening.csv`
- Used strictly for educational and learning purposes

## 📊 Workflow

1. Load the resume dataset  
2. Perform text preprocessing and cleaning  
3. Extract features using TF-IDF  
4. Train a Random Forest classifier  
5. Evaluate model performance  
6. Predict and classify resumes
